PROJECTS AND EXTRACURRICULARS
Deep Learning - MNIST dataset

In fall of 2017, I took on a deep learning project that was recommended to me by my professor. A classmate and I were able to develop a Java program that could recognize handwritten digits by training and testing through neural networks. Through this project, we were able to learn a lot about artificial intelligence, deep learning, and neural networks. These fascinating fields have endless applications, and it was a great experience to be able to work on a “gateway” project to these fields. Neural networks, being similar in function and purpose to our brain, have numerous current and potential applications, including, but not limited to, chemical engineering, nuclear power plants, wind speed energy modeling, and construction. Not only that, but there are current uses for neural networks too, such as processing the bank account number or other numerical information from handwritten checks or forms. The future is bright for neural networks, as they will become a major focus of research in computer science going forwards. We were able to get a head start on neural networks by finishing this project while still in undergraduate college.

Home Automation (Arduino and RaspberryPi)

During winter and spring of 2018, I worked on a home automation project as part of my STEM cohort at Bellevue College with another class member. We used Arduino and RaspberryPi to create a system that connected to different features using Wifi communication. We were able to create a user interface that could control features such as lights, smoke sensors, temperature, etc.
